# BIO 361 Gluconeogenesis Quiz

This is a simple static quiz app you can upload to GitHub and run with GitHub Pages.

## Files
- `index.html`
- `style.css`
- `questions.js`
- `script.js`

## How to use locally
Open `index.html` in your browser.

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save.
7. GitHub will give you a public link for your quiz.

## Features
- 42 multiple choice questions
- optional question shuffle
- optional answer choice shuffle
- score report
- wrong answer review
